## So how to compile?
### 1. Extract btree library and find 'libbt.a' in bt-5.0.0/lib 
   Replace `libbt.a` with your `libbt.a` 

### 2. Create dictionary db from demo31.c
Note: This includes removing used databases
>> ./createdict.sh

### 3. Run dictionary
>> ./rundict.sh
